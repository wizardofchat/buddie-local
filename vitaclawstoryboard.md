## HOW TO ASSEMBLE IN CAPCUT (step by step)

1. Open CapCut → New Project → import all 5 clips
2. Arrange in order: A → B → C → D → E
3. Trim each clip to target duration (use storyboard above)
4. Add black screen at start (0-5s) — tap + → blank
5. Add cream screen at end (57-60s) — tap + → blank, set background colour #F2EFE8
6. Add captions: tap Text → Auto Captions OR manually add each line from master list
7. Font: clean sans-serif (Montserrat or similar) — white on dark, near-black on cream
8. No music needed — silence is more credible for a technical demo
9. Export: 1080x1920 (portrait) for X mobile

## HONEST NOTES

- Don't show code. Non-technical audience will scroll past.
- Do show the Telegram message arriving — that's the most human moment.
- The £0.04/day stat is your strongest hook — use it early or in the X copy.
- Portrait format outperforms landscape on X mobile feed.
- No voiceover needed — captions are enough and work on silent autoplay.
